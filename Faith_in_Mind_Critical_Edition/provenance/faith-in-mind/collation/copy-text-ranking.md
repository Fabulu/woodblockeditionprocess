# Faith in Mind Copy-Text Candidate Ranking

Date: 2026-04-14  
Status: pre-OCR ranking  
Scope: `T1-T5` and `A1-A3`

This note ranks the current copy-text candidates before OCR and direct collation.

## Ranking criteria

The ranking below uses only what is currently secure:

1. directness to `信心銘`
2. witness type
3. likely textual closeness versus derivative status
4. validation quality and local usability
5. practical suitability as an OCR / transcription spine

This is a working ranking, not a final genealogical claim.

## Ranked candidates

### 1. `T1` `三祖大師信心銘`

- Folder: `Faith_in_Mind_Korea_Commons`
- Why it ranks first:
  - only standalone direct witness in the locked core
  - not embedded inside a larger anthology branch
  - locally validated, stable, and directly usable
  - best candidate for a base-text spine because it minimizes anthology-boundary ambiguity
- Current recommendation:
  - use as the initial working copy-text

### 2. `T4` `四部録` (`1629`)

- Folder: `Faith_in_Mind_Kyoto_RB00009461_Shiburoku`
- Why it ranks second:
  - distinct early Kyoto `四部録` printing
  - likely strong anthology witness for the transmitted text
  - image-manifest witness is complete and stable
  - useful as the first major control against `T1`
- Limitation:
  - anthology witness, so extraction boundaries must be confirmed

### 3. `T5` `四部録` (`1631`)

- Folder: `Faith_in_Mind_Kyoto_RB00012929_Shiburoku`
- Why it ranks third:
  - another distinct Kyoto `四部録` printing
  - gives a second strong anthology control within the same broad branch
  - complete and stable local image set
- Limitation:
  - anthology witness, not standalone

### 4. `T2` `四部録`

- Folder: `Sengcan_Faith_in_Mind_NDL_Shibulu`
- Why it ranks fourth:
  - directly names `三祖鑑智禪師信心銘`

- Why below `T4/T5`:
  - printing relationship to the Kyoto `四部録` witnesses is unresolved
  - less attractive than the Kyoto pair as a first anthology control family

### 5. `T3` `入衆日用`

- Folder: `Faith_in_Mind_Sengcan_Kyoto_RB00016909`

### 6. `A3` `四部録抄`
