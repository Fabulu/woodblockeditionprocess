# Agent 6 End-Matter Audit

Scope: witness-only audit of late end-matter / colophon leaves, with emphasis on recoverable dates, names, and non-running-text classifications. No external text was used.

## Main findings

- The late sequence is layered, not a single colophon block. It includes a postscript/sign-off, a clear termination leaf, endorsement or `跋` material, later notes, printing-history matter, a later print colophon, then blank / cover surfaces.
- Several current low-confidence labels undercall what is actually recoverable from the witness.
- The main classification error to avoid is treating seal-heavy or cover leaves as simply `〔blank / no legible text recovered〕`.

## Classification corrections to carry forward

- If any late leaf is currently summarized as `〔blank / no legible text recovered〕`, `PDF p.052 right` should not be. It is a seal-obscured text leaf.
- `PDF p.054 right` should not be reduced to seals-only. It has clear date/place/signature text plus seals.
- `PDF p.057 right` should be treated as cover embossing, not as recoverable text and not as an ordinary blank leaf.
- `PDF p.055 left` can remain the model of a true blank late leaf.
